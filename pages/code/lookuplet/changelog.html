Release notes

Release 1.2 (17-Mar-2002):

  • Rewrote the whole thing in Python (using Glade for the UI).
  • Added history support along with tab completion from the history.
  • Convert newlines to spaces when grabbing the selection. If we think that the selection is a URL (it starts with http: etc.), then we remove spaces as well.

Release 1.1 (17-Aug-2001):

  • Fixed support for running as a GNOME applet and made the code automatically figure out whether or not do so.
  • Sorted out building things as a Debian package.

Release 1.0 (16-Aug-2001):

  • Initial release.
